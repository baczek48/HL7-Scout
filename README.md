# HL7 Scout

Parser i edytor wiadomosci HL7 v2.x z wysylka MLLP.

## Wersje

### hl7_scout/
Wersja podstawowa — parser, edytor segmentow, wysylka MLLP.

### hl7_scout_db/
Wersja rozszerzona — wszystko co wyzej + polaczenie z baza Oracle, wyszukiwanie paczek HL7.

## Wymagania
- Python 3.11+
- PyQt6
- oracledb (tylko wersja DB)
